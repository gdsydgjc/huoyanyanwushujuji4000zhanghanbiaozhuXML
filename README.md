# 火焰烟雾数据集4000张含标注XML

## 简介

本仓库提供了一个包含4000张火焰和烟雾图像的数据集，每张图像均附带有标注的XML文件。该数据集适用于火焰和烟雾检测相关的研究与开发工作。

## 数据集内容

- **图像数量**: 4000张
- **图像格式**: JPEG
- **标注文件**: 每张图像对应一个XML文件，标注了火焰和烟雾的位置信息。

## 使用场景

该数据集适用于以下场景：

- 火灾预警系统的训练与测试
